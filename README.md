# メルカリ便 送料最安チェッカー

メルカリの発送方法、どれが一番安いかすぐ分かるツールです。

👉 https://hizumippc.github.io/Mercari-Shipping-Cost-Checker/

---

## 🔍 概要

発送サイズ・厚さ・重さを選ぶだけで、  
利用可能な配送方法の中から**最安の上位3つ**を表示します。

「どれで送るのが一番安いの？」を一発で解決。

---

## 🛠 主な機能

- 厚さ / サイズ / 重さで絞り込み
- 利用可能な配送方法のみ表示
- 送料が安い順に自動ソート
- 上位3件をランキング表示（🥇🥈🥉）
- 各配送方法の詳細表示
  - 送料内訳（専用箱含む）
  - サイズ・重量制限
  - 発送場所

---

## 📦 対応配送方法

- らくらくメルカリ便
  - ネコポス
  - 宅急便コンパクト
  - 宅急便（60〜200サイズ）

- ゆうゆうメルカリ便
  - ゆうパケットポストmini
  - ゆうパケットポスト
  - ゆうパケット
  - ゆうパケットプラス
  - ゆうパック（60〜170サイズ）

- エコメルカリ便
  - 宅配便
  - 置き発送

- 梱包・発送たのメル便

---

## 💡 このツールを作った理由

発送方法が多すぎて、

- 結局どれが一番安いのか分からない
- サイズごとに調べるのが面倒

と感じたため。

**「とにかく最安だけ知りたい」人向け**に作りました。

---

## 🚀 使い方

1. 厚さを選択
2. サイズ（3辺合計）を選択
3. 重さを選択

→ 自動で結果が更新されます

---

## ⚠️ 注意事項

- 実際の発送時は必ず正確なサイズ・重量を測定してください
- ポスト投函は実際の投函可否に依存します（目安3cmなど）
- 送料・仕様は変更される可能性があります
